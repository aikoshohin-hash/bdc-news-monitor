Methodology

  • 対象: プライベートクレジット／BDC（Business Development Company）に関する国内外のニュース
  • 期間: 2024-01-01〜現在（GitHub Actions 毎日 07:00 JST 更新）
  • 関連性判定: MUST/SHOULD/EXCLUDE の 3 層キーワードゲート（config/keywords.yaml）
  • センチメント: 完全オフライン — 英語は Loughran-McDonald 金融極性辞書、日本語は独自辞書＋oseti、フレーズ単位ドメイン・オーバーライド＋BDCドメイン辞書オーバーレイ（config/sentiment_overrides_bdc.yaml）。スコアは tanh((pos-neg)/tokens × 6) のS字カーブで [-1, +1] の連続値。外部 API は使用しない
  • ピア相対センチメント (Issue #8): z = (ticker_sent - peer_mean) / peer_std を30日移動平均で平滑化。ピア集合 = BIZD 上位13社
  • 株価: yfinance（Yahoo Finance）からの日次終値。BIZD 構成上位 13 社＋ベンチマーク（BIZD, ^GSPC, HYG）
  • 四半期メトリクス: SEC EDGAR XBRL companyfacts API（無償・無認証）から NAV/share、総投資額（時価）、NII/share、配当/株、Asset Coverage 等を抽出
  • ヒートマップ (Issue #7): BDC × 日付のセルに日次センチメント平均を colorblind-safe diverging パレットで表示。過去90/180/365日切替
  • アラート (Issue #6): config/alert_rules.yaml に定義されたルールに基づき、GitHub Actions の日次バッチ最終段で条件評価 → Slack webhook / Email 通知。24時間クールダウン

データソース

  • Google News RSS（英／日キーワード検索）
  • GDELT 2.0 Article Search API（海外広範）
  • SEC EDGAR Full-Text Search（N-2, 10-K, 10-Q, 8-K）
  • 任意 RSS フィード（config/sources.yaml で管理）
