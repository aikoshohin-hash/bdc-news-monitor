Methodology

  • 対象: プライベートクレジット／BDC（Business Development Company）に関する国内外のニュース
  • 期間: 2024-01-01〜現在（GitHub Actions 毎日 07:00 JST 更新）
  • 関連性判定: MUST/SHOULD/EXCLUDE の 3 層キーワードゲート（config/keywords.yaml）
  • センチメント: 完全オフライン — 英語は Loughran-McDonald 金融極性辞書、日本語は独自辞書＋oseti、フレーズ単位ドメイン・オーバーライド。外部 API は使用しない
  • 株価: yfinance（Yahoo Finance）からの日次終値。BIZD 構成上位 12 社＋ベンチマーク（BIZD, ^GSPC, HYG）
  • 重ね描き: 2024-01-01 を 100 とした指数化切替、20d/60d 移動平均オーバーレイ

データソース

  • Google News RSS（英／日キーワード検索）
  • GDELT 2.0 Article Search API（海外広範）
  • SEC EDGAR Full-Text Search（N-2, 10-K, 10-Q, 8-K）
  • 任意 RSS フィード（config/sources.yaml で管理）
